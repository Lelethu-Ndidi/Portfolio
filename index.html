My Services

Front-end Developer

Front-end developers are responsible for a website’s user-facing code and the architecture of its immersive user experiences, which run on the open web platform or act as compilation input for non-web platform environment. In order to execute those objectives, front-end devs must be adept at three main languages: HTML, CSS, and Javascript programming. And I have done all 3 and I am very familiar with them.

Back-End Deloper

The back end runs a website—the user doesn’t see it or interact with it, but it is always running in the background, delivering functionality and a desktop-like experience. Back-end developers play a critical role in web development teams, and make sure that data or services requested by the front end system or software are delivered. For this, I have only done python, flask and mysql for the database.
